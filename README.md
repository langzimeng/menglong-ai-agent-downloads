# 梦龙AI-Agent 下载

本仓库仅用于发布梦龙AI-Agent成品，不包含源代码。

## macOS Apple Silicon

适用于 M1、M2、M3、M4 系列 Mac，应用内置 ARM64 Python，无需单独安装 Python。

请从 [Releases](https://github.com/langzimeng/menglong-ai-agent-downloads/releases/latest) 下载 `梦龙AI-Agent-macOS-Apple-Silicon.zip`。

由于当前版本未进行 Apple Developer 公证，首次运行时请将应用拖入“应用程序”，然后在终端执行：

```bash
sudo xattr -dr com.apple.quarantine "/Applications/梦龙AI-Agent.app"
open "/Applications/梦龙AI-Agent.app"
```
